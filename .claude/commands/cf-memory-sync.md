# Claude Flow Memory Sync

Synchronize memory across instances.

## Usage
```
/cf-memory-sync [target]
```

## Instructions

Use the `mcp__claude-flow__memory_sync` tool:

1. **Required Parameters:**
   - `target`: Target instance or endpoint to sync with

## Purpose

Replicates memory data across multiple Claude Flow instances for distributed setups.

## Example

```
Sync to remote instance:
- target: "remote-instance-url"
```

Execute: `mcp__claude-flow__memory_sync` with target="..."
