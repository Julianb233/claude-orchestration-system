# SM - Social Media Shortcut

Alias for `/social`. See `/social` for full documentation.

## Quick Reference

| Command | Action |
|---------|--------|
| `/sm create "..."` | Create multi-platform posts |
| `/sm calendar` | View content calendar |
| `/sm preview` | Generate HTML mockups |
| `/sm schedule` | Schedule to Notion |
| `/sm export buffer` | Export for Buffer |
| `/sm drafts` | View current drafts |
| `/sm ideas [topic]` | Generate post ideas |

## Execution

This command redirects to `/social` with the same arguments.

When invoked, execute the `/social` command with any provided arguments.
