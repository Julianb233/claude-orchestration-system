# Scripts Shortcut

Shortcut for `/script-writer`. Produces branded social media scripts.

## Quick Usage

- `/scripts full ClientName "topic" 30sec` - Full research + script
- `/scripts quick ClientName "topic" 60sec` - Quick script (existing client)
- `/scripts batch ClientName 5 "series theme" 15sec` - Multiple scripts
- `/scripts schedule ClientName "topic" 30sec 2024-12-20 14:00` - Schedule to calendar

## Duration Formats

- `15s`, `30s`, `60s` - Seconds
- `2m`, `5m`, `15m`, `30m` - Minutes
- `1h` - Hour

## Requirements

$ARGUMENTS
