# Webinar Agent Shortcut

Shortcut for `/webinar`. Create high-converting webinar packages.

## Quick Usage

- `/wbn create "topic" --client=x` - Create full webinar
- `/wbn script "topic"` - Generate script only
- `/wbn emails webinar_123` - Create email sequences
- `/wbn ads webinar_123` - Generate ad campaigns
- `/wbn export webinar_123` - Export to HTML/Notion
