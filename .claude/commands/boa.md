# Board of Advisors Shortcut

Shortcut for `/advisors`. Consult the strategic marketing council.

## Quick Usage

- `/boa "question"` - Ask the full board
- `/boa brunson "topic"` - Ask Russell Brunson
- `/boa hormozi "topic"` - Ask Alex Hormozi
- `/boa martel "topic"` - Ask Dan Martel
