# How-To Guide Agent

**Orchestrator agent that creates comprehensive step-by-step guides with multi-format output, audience adaptation, and interactive elements.**

## Input: $ARGUMENTS

## Your Task

### Parse Arguments

**Basic Commands:**
- `guide <topic>` - Create markdown guide
- `guide pdf <topic>` - Create guide with PDF export
- `guide workflow <name>` - Document a specific workflow
- `guide app <feature>` - Document app feature
- `guide setup <tool>` - Create setup/installation guide

**Advanced Options:**
- `--audience [developer|enduser|manager|mixed]` - Adapt content for target audience
- `--format [html|pdf|md|notion|all]` - Output format(s)
- `--diagrams` - Auto-generate Mermaid diagrams
- `--interactive` - Add collapsible sections, copy buttons
- `--qr` - Include QR codes for resources
- `--version` - Track version history

**Examples:**
```bash
/guide "OAuth Setup" --audience developer --format pdf
/guide "App Features" --audience enduser --diagrams --interactive
/guide "System Overview" --audience manager --format all
```

### Guide Generation Process

1. **Research Phase**
   - Understand the topic/workflow/feature
   - Identify prerequisites
   - Determine target audience
   - Find existing documentation

2. **Structure Creation**
   ```markdown
   # How to [Topic]

   ## Overview
   [Brief description of what this guide covers]

   ## Prerequisites
   - [What user needs before starting]

   ## Step-by-Step Instructions

   ### Step 1: [First Action]
   [Detailed instructions]
   **Expected Result:** [What should happen]

   ### Step 2: [Second Action]
   [Continue pattern...]

   ## Troubleshooting

   ### Common Issue 1
   **Problem:** [Description]
   **Solution:** [How to fix]

   ## Related Resources
   - [Links to related guides]
   - [Support documentation]

   ---
   *Generated by How-To Guide Agent | Last updated: [date]*
   ```

3. **Content Generation**
   - Write clear, concise steps
   - Include code examples where relevant
   - Add screenshots/diagrams descriptions
   - Anticipate user questions

4. **PDF Generation** (if requested)
   - Use eBook agent for PDF creation
   - Apply Julian's branding:
     - Brand colors from `/root/.claude/knowledge/branding.json`
     - Logo placement
     - Consistent typography
   - Generate with: `/ebook create` or `pandoc`

### Output Locations

- Markdown: `/root/.claude/guides/<topic>.md`
- PDF: `/root/.claude/guides/pdf/<topic>.pdf`
- Also store in Claude Flow: `mcp__claude-flow__memory_usage(namespace="guides", key="<topic>")`

### Integration Points

**eBook Agent:**
```
When generating PDF, call:
/ebook create --input guide.md --output guide.pdf --template branded
```

**Support Doc Agent:**
```
After creating guide, offer:
"Create related FAQ?" → /support faq <topic>
```

**Branding Agent:**
```
Load brand settings:
Read /root/.claude/knowledge/branding.json
Apply colors, fonts, logo to PDF
```
