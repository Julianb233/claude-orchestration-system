# Claude Flow Memory Compress

Compress memory data to optimize storage.

## Usage
```
/cf-memory-compress [namespace]
```

## Instructions

Use the `mcp__claude-flow__memory_compress` tool:

1. **Optional Parameters:**
   - `namespace`: Specific namespace to compress (compresses all if not specified)

## Purpose

Optimizes storage space by compressing memory entries. Useful for large datasets or long-running sessions.

## Example

```
Compress project namespace:
- namespace: "projects"
```

Execute: `mcp__claude-flow__memory_compress` with namespace="projects"
