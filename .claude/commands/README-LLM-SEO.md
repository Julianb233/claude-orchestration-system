## Key LLM SEO Principles

### What Makes Content Rank in LLMs:

1. **Answer First** - Put the answer in the first 2 sentences
2. **Statistics** - Include 10+ verifiable data points
3. **Expert Quotes** - 2-3 credentialed expert validations
4. **Clear Structure** - H1→H2→H3 hierarchy
5. **Q&A Format** - Questions as headings
6. **Authority Signals** - Reviews, backlinks, platform presence
7. **Technical Health** - Fast pages, schema markup, mobile-friendly

### What Doesn't Work:
- ❌ Keyword stuffing
- ❌ llms.txt files (minimal adoption)
- ❌ FAQ schema alone (content quality matters more)
- ❌ Ignoring Google rankings (still the prerequisite)
