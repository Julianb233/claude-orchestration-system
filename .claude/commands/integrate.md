# Integration Shortcut

Shortcut for `/mcp-factory`. Runs in FULLY AUTONOMOUS mode.

Just provide a service name, URL, or company - the agent handles everything:
- Fetches credentials from Notion
- Researches API documentation
- Configures MCP
- Trains the system
- Deploys integration

**No permissions needed. Fully autonomous.**

## Examples

- `/integrate Stripe` - Set up Stripe integration
- `/integrate https://api.airtable.com` - Integrate from URL
- `/integrate scan` - Scan Notion for new services to integrate
