# Award-Winning Website Design Checklist

**Use this checklist when creating or auditing websites. Reference by Bubba and all agents during website projects.**

---

## VISUAL DESIGN & AESTHETICS (15 Items)

### Typography
1. **Consistent Font Pairing** - 1-2 fonts max (serif headers, sans-serif body)
2. **Optimal Line Length** - 45-75 characters per line
3. **Proper Font Sizing** - Base 16px minimum (18px ideal)
4. **Legible Font Choices** - Proven typefaces (Inter, Roboto, Georgia)
5. **Heading Hierarchy** - Clear H1→H2→H3 proportional structure

### Color & Contrast
6. **High Contrast Ratios** - 4.5:1 minimum (WCAG AA)
7. **Limited Color Palette** - 3-5 primary colors + neutrals
8. **Intentional Color Psychology** - Colors match brand message
9. **Accessible Color Schemes** - Avoid red-green combinations
10. **Brand Color Consistency** - Same hex values everywhere

### Layout & Whitespace
11. **Abundant Whitespace** - 30%+ negative space
12. **Clear Visual Hierarchy** - Important elements largest/highest contrast
13. **Symmetrical Grid System** - Consistent 8px/12px/16px grid
14. **Consistent Margins & Padding** - Spacing scale (8, 16, 24, 32px)
15. **Responsive Breakpoints** - Works at 320px, 768px, 1024px, 1440px+

---

## UI/UX & INTERACTION (22 Items)

### Navigation
16. **Sticky Header Navigation** - Remains accessible on scroll
17. **Clear Navigation Labels** - Specific labels, not vague
18. **Visible Current Page Indicator** - Active state highlighted
19. **Navigation Hierarchy** - Primary separated from secondary
20. **Breadcrumb Navigation** - Shown on inner pages
21. **Mobile Menu Usability** - Intuitive hamburger/drawer

### Interactions & Micro-interactions
22. **Smooth Hover States** - Clear feedback on hover
23. **Button Design** - Looks clickable, sufficient padding
24. **Loading States** - Progress indicators for >2s loads
25. **Form Validation Feedback** - Real-time, plain language errors
26. **Scroll Animations** - Subtle fade/slide on scroll
27. **Page Transitions** - Smooth between pages
28. **Cursor Feedback** - Changes on interactive elements

### Forms & Input
29. **Single-Column Forms** - Avoid multi-column
30. **Inline Labels Above Fields** - Always visible
31. **Visible Placeholder Text** - Doesn't disappear
32. **Inline Help Text** - Clarification below fields
33. **Smart Input Types** - Email keyboard for email, etc.
34. **One Action Per Form** - Single primary button
35. **Form Spacing** - 24px+ between fields
36. **Required Field Indicators** - Asterisk or text
37. **Success Confirmation** - Clear after submission

### Accessibility (WCAG 2.1 AA)
38. **Keyboard Navigation** - Tab key access with focus states
39. **ARIA Labels** - Screen readers understand controls
40. **Color Not Only Indicator** - Also icon/text for info
41. **Image Alt Text** - Descriptive for all images
42. **Heading Structure** - Semantic H1-H6 hierarchy
43. **Skip Links** - "Skip to main content" visible on focus
44. **Focus Indicators** - Visible focus rings
45. **Motion & Animation** - Respect prefers-reduced-motion

### Responsive Design
46. **Mobile-First Approach** - Mobile version first
47. **Touch-Friendly Targets** - 44x44px minimum
48. **Responsive Images** - Scale properly at breakpoints
49. **Readable Mobile Text** - 16px+ on mobile
50. **Flexible Layouts** - Flexbox/grid, no fixed widths

---

## TECHNICAL EXCELLENCE (15 Items)

### Performance (Core Web Vitals)
51. **LCP < 2.5s** - Fast main content display
52. **FID < 100ms** - Responsive to interaction
53. **CLS < 0.1** - No unexpected layout shifts
54. **FCP < 1.8s** - Fast first paint
55. **Optimized Images** - WebP format, lazy loading
56. **Minified CSS/JS** - Remove unnecessary characters
57. **Code Splitting** - Load only needed code
58. **Caching Strategy** - 30+ days for static assets
59. **CDN Usage** - Globally distributed delivery

### SEO Fundamentals
60. **Semantic HTML** - Proper HTML5 tags
61. **Unique Meta Descriptions** - 120-160 characters
62. **Canonical Tags** - Prevent duplicate content
63. **Mobile-Friendly Design** - Passes Google test
64. **Schema Markup** - Structured data for rich snippets
65. **Fast Page Speed** - Core Web Vitals compliant
66. **SSL Certificate** - HTTPS required
67. **Descriptive URLs** - Reflect content structure

### Code Architecture
68. **Semantic HTML Structure** - Properly organized
69. **CSS Organization** - Modular, maintainable
70. **Responsive Framework** - Consistent grid system
71. **Accessibility-First Coding** - Built in from start

---

## CONVERSION & TRUST (15 Items)

### Above-the-Fold Content
72. **Clear Value Proposition** - Understand in 1-2 seconds
73. **Compelling Hero Section** - Quality image/video + headline + CTA
74. **Trust Elements Visible** - Logo, badges, awards
75. **Clear Navigation Focus** - Users know how to explore

### Calls-to-Action (CTAs)
76. **Prominent CTA Buttons** - Contrasting color, 44x44px min
77. **Action-Oriented CTA Copy** - "Get Started" not "Submit"
78. **Multiple Strategic CTAs** - At journey touchpoints
79. **CTA Button Affordance** - Looks clickable
80. **Authentic Scarcity/Urgency** - Only when genuine

### Social Proof & Trust
81. **Customer Testimonials** - 3-5 with names/photos/titles
82. **Review Ratings** - Star ratings prominent
83. **Client Logos** - Well-known clients displayed
84. **Case Studies** - Detailed success stories
85. **User-Generated Content** - Customer photos/videos
86. **Trust Badges** - Security seals, certifications

### Emotional Design
87. **Brand Storytelling** - Mission, values communicated
88. **Authentic Imagery** - Real photos, not stock
89. **Human-Centered Design** - People using products
90. **Clear Privacy Policy** - Easy to find
91. **Transparent Pricing** - No hidden fees

---

## CONTENT & FLOW (12 Items)

### Information Architecture
92. **Logical Site Structure** - Intuitive content organization
93. **Scannable Content** - Short paragraphs, bullets, subheadings
94. **Progressive Disclosure** - Key info first, details hidden
95. **Related Content Links** - "You might also like"

### Copywriting
96. **Benefit-Focused Headlines** - Lead with value
97. **Short Sentences** - 15-17 words average
98. **Active Voice** - Direct, engaging
99. **Clarity Over Cleverness** - No jargon
100. **Problem-Agitate-Solve** - Proven persuasion model
101. **Consistent Tone of Voice** - Brand personality
102. **CTA Copy Clarity** - Explain what happens next
103. **Data & Stats** - Specific metrics for credibility

### Visual Flow
104. **F-Pattern Layout** - Match eye tracking
105. **Z-Pattern for Simple Layouts** - Diagonal flow
106. **Directional Cues** - Images pointing to CTAs
107. **Consistent Visual Language** - Icons, colors match
108. **Emphasis Through Contrast** - Important stands out

## QUICK AUDIT SCORING

Rate each item 0-3:
- 0 = Missing/broken
- 1 = Partially implemented
- 2 = Implemented but needs improvement
- 3 = Excellent implementation

**Score Interpretation:**
- 280-324: Award-winning quality
- 220-279: Production-ready
- 160-219: Needs significant work
- Below 160: Major overhaul required
