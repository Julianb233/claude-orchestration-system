## PART 1: Introduction (0-15 min)

### Hook (First 30 seconds)

**Pattern Interrupt:**
```
"What if I told you that _______________?"
```

**Big Promise:**
```
"In the next 60 minutes, I'm going to show you how to
_____________ without _____________ - even if _____________."
```

### Build Rapport

**Your Story (Epiphany Bridge):**
- Your backstory: _________________________
- The wall you hit: _________________________
- Your epiphany moment: _________________________
- The transformation: _________________________

**Credibility (quick, not braggy):**
- Results achieved: _________________________
- People helped: _________________________
- Media/recognition: _________________________

### Set the Frame

**What This Is:**
```
"This is NOT a _____________ [what they expect].
This IS a _____________ [what it actually is]."
```

**Rules:**
1. Stay until the end (special bonus)
2. Take notes (engaged learning)
3. Ask questions in chat

## PART 4: The Close (55-75 min)

### Create Urgency (REAL)

**Scarcity:**
```
"This offer is only available to _____ people because _____________."
```

**Urgency:**
```
"You need to decide now because _____________."
```

### Handle Objections

**Price Objection:**
```
"I know $_______ might seem like a lot, but think about it...
[Reframe the investment]"
```

**Time Objection:**
```
"You might be thinking you don't have time, but...
[Address time concern]"
```

**Skepticism Objection:**
```
"Maybe you've tried things before and they didn't work...
[Explain why this is different]"
```

### Call to Action

```
"Go to [URL] right now and click the button.
You'll fill out a short application and _____________.

Do it now because _____________."
```

### Recap Stack One More Time

```
"Remember, you're getting:
✓ [Item 1] (Worth $_____)
✓ [Item 2] (Worth $_____)
✓ [Item 3] (Worth $_____)
✓ [Bonus 1] (Worth $_____)
✓ [Bonus 2] (Worth $_____)

Total value: $_______

Your investment: Just $_______

Plus my [Guarantee Type] guarantee.

Go to [URL] now."
```

## Post-Webinar Sequence

### Email 1: Replay (Send immediately)
- Subject: "Did you miss this?"
- Content: Replay link + recap of offer

### Email 2: Case Study (Day 2)
- Subject: "[Name] got [Result]"
- Content: Success story + link to offer

### Email 3: FAQ (Day 3)
- Subject: "Questions about [Offer]?"
- Content: Common questions answered

### Email 4: Final Warning (Day 4)
- Subject: "Closing tonight"
- Content: Last chance + full stack recap

### Email 5: Closed (Day 5)
- Subject: "Sorry, you missed it"
- Content: Offer closed + waitlist for next time

## Webinar Checklist

### Before:
- [ ] Title creates curiosity
- [ ] 3 Secrets mapped to false beliefs
- [ ] Epiphany bridge stories ready
- [ ] Stack slide prepared
- [ ] Guarantee clear
- [ ] Urgency/scarcity is REAL
- [ ] CTA URL tested
- [ ] Slides designed
- [ ] Tech tested (audio, video, screen share)

### During:
- [ ] Start with pattern interrupt
- [ ] Build rapport quickly
- [ ] Deliver secrets with stories
- [ ] Stack value visually
- [ ] Price reveal with anchoring
- [ ] Handle objections
- [ ] Clear CTA repeated 3+ times
- [ ] Stay for Q&A

### After:
- [ ] Send replay immediately
- [ ] Follow-up sequence loaded
- [ ] Close cart at deadline
- [ ] Analyze metrics
- [ ] Collect testimonials

## Quick Reference: The 4-Step Formula

1. **Introduction:** Build rapport, share story, set frame
2. **Content:** Break 3 false beliefs with epiphany bridges
3. **Stack:** Build value tower, reveal price with anchor
4. **Close:** Urgency, objections, clear CTA
