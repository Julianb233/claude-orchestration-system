## Framework Deep Dives

### Brunson Frameworks

#### Secret Formula (Start Here)
**Use when:** Starting any new offer, funnel, or campaign

4 Questions:
1. Who is your dream customer?
2. Where are they congregating?
3. What bait will attract them?
4. What result do you want?

#### Value Ladder
**Use when:** Building offers, increasing LTV, creating ascension

```
                    ▲
                   /│\
                  / │ \ High Ticket
                 /  │  \
                /   │   \ Mid Ticket
               /    │    \
              /     │     \ Low Ticket
             /      │      \
            /       │       \ Lead Magnet
           ─────────┴─────────
                  FREE
```

#### Hook-Story-Offer
**Use when:** Writing any marketing content, emails, ads, videos

1. HOOK: Pattern interrupt + curiosity
2. STORY: Epiphany bridge + relatability
3. OFFER: Stack value + create urgency

#### Attractive Character
**Use when:** Building personal brand, creating content, writing emails

Four Archetypes:
- **Leader:** Shows the path they've already walked
- **Adventurer:** Takes audience on journey with them
- **Reporter:** Interviews experts, shares findings
- **Reluctant Hero:** Didn't want spotlight but felt called

#### Perfect Webinar
**Use when:** Selling high-ticket via presentation

Structure:
1. Introduction (build rapport, share results)
2. Content (3 secrets/beliefs to break)
3. Stack (build value tower)
4. Close (urgency, scarcity, guarantee)

### Hormozi Frameworks

#### Value Equation
**Use when:** Pricing, offer creation, sales conversations

```
                Dream Outcome × Perceived Likelihood
VALUE = ─────────────────────────────────────────────
                   Time Delay × Effort Required
```

Levers to pull:
- ↑ Dream Outcome: Make it bigger, more specific
- ↑ Likelihood: Proof, guarantees, credentials
- ↓ Time Delay: Faster results, quick wins
- ↓ Effort: Done-for-you, automation, simplicity

#### Grand Slam Offer
**Use when:** Creating or improving any offer

Steps:
1. Dream outcome (what they REALLY want)
2. List ALL problems (before/during/after)
3. Solutions for each problem
4. Delivery vehicles (DFY/DWY/DIY)
5. Stack bonuses
6. Add guarantee
7. Name it powerfully

#### Lead Magnet Framework
**Use when:** Creating lead magnets, free offers

Criteria:
- Solves ONE complete problem
- Immediate value
- Minimal consumption time
- Naturally leads to paid offer
- Easy to deliver at scale

#### Advertising Equation
**Use when:** Evaluating ad performance, scaling

```
ROAS = (LTV × Conversion × Volume) / CAC
```

Fix in order:
1. LTV (retention, upsells)
2. Conversion (landing pages, offer)
3. CAC (targeting, creative)
4. Volume (scale what works)

### Martell Frameworks

#### Buyback Principle
**Use when:** Making ANY hiring decision

> "Don't hire to grow your business. Hire to buy back your time."

#### Buyback Rate
**Use when:** Deciding what to delegate

```
             Annual Revenue Goal
Rate = ─────────────────────────
            2,000 hours/year
```

Rule: Anything below your rate should be delegated.

#### DRIP Matrix
**Use when:** Auditing how you spend time

```
          HIGH ENERGY
              │
    INVEST    │   PRODUCE
  (hobby-like │ (your genius
   activities)│    zone)
              │
LOW ──────────┼────────── HIGH
VALUE         │          VALUE
              │
   DELEGATE   │   REPLACE
 (outsource   │ (train someone
   ASAP)      │    to do)
              │
          LOW ENERGY
```

#### Replacement Ladder
**Use when:** Deciding WHO to hire

Order (bottom to top):
1. Executive Assistant (FIRST!)
2. Virtual Assistants / Admin
3. Project Managers / Team Leads
4. Department Heads
5. Executive Team

#### Camcorder Method
**Use when:** Preparing to delegate any task

Steps:
1. Record yourself doing it
2. Create SOP from recording
3. Test with new person
4. Iterate based on questions
5. Finalize documentation
