## Core Frameworks

### The Buyback Principle

> "Don't hire to grow your business. Hire to buy back your time."

**The Shift:**
- OLD: "I need to make more money to hire people"
- NEW: "I need to hire people to make more money"

**Why It Works:**
- Your time is your most valuable asset
- Every hour spent on low-value work costs you high-value opportunities
- The right hires pay for themselves through freed capacity

---

### The Buyback Rate

Calculate what one hour of your time is truly worth:

```
                    Annual Revenue Goal
Buyback Rate = ─────────────────────────
                    2,000 hours/year
```

**Example:**
- Goal: $500,000/year
- Buyback Rate: $250/hour

**Rule:** Anything that can be done for LESS than your Buyback Rate should be delegated.

**Application:**
| Your Buyback Rate | Delegate Tasks Under |
|-------------------|---------------------|
| $50/hour | $25/hour tasks |
| $100/hour | $50/hour tasks |
| $250/hour | $125/hour tasks |
| $500/hour | $250/hour tasks |

### The Buyback Loop

A continuous process for reclaiming your time:

```
┌─────────────────────────────────────────────┐
│                                             │
│    ┌─────────┐                              │
│    │  AUDIT  │ ← List everything you do     │
│    └────┬────┘                              │
│         │                                   │
│         ▼                                   │
│    ┌─────────┐                              │
│    │TRANSFER │ ← Move low-value tasks       │
│    └────┬────┘                              │
│         │                                   │
│         ▼                                   │
│    ┌─────────┐                              │
│    │  FILL   │ ← Fill with high-value work  │
│    └────┬────┘                              │
│         │                                   │
│         └──────────────────────────────────→│
│                     REPEAT                  │
└─────────────────────────────────────────────┘
```

**Step 1: AUDIT**
- Track every task for 2 weeks
- Note time spent on each
- Rate each task: energizes vs. drains

**Step 2: TRANSFER**
- Identify tasks below your Buyback Rate
- Document processes
- Hire or delegate to existing team

**Step 3: FILL**
- Use reclaimed time for high-value activities
- Focus on $10K/hour activities
- Repeat the loop

### The DRIP Matrix

Categorize every task by value and energy:

```
                    HIGH ENERGY
                         │
           ┌─────────────┼─────────────┐
           │             │             │
           │  INVEST     │   PRODUCE   │
           │             │             │
           │ High energy │ High energy │
           │ Low $ value │ High $ value│
           │             │             │
   LOW ────┼─────────────┼─────────────┼──── HIGH
   VALUE   │             │             │     VALUE
           │             │             │
           │  DELEGATE   │   REPLACE   │
           │             │             │
           │ Low energy  │ Low energy  │
           │ Low $ value │ High $ value│
           │             │             │
           └─────────────┼─────────────┘
                         │
                    LOW ENERGY
```

| Quadrant | Definition | Action |
|----------|------------|--------|
| **DELEGATE** | Low value, drains you | Outsource immediately |
| **REPLACE** | High value, drains you | Train someone ASAP |
| **INVEST** | Low value, energizes you | Keep but limit time |
| **PRODUCE** | High value, energizes you | Do more of this! |

### The 5 Time Assassins

Self-sabotaging behaviors that kill productivity:

**1. The Staller**
- Waits until perfect before starting
- Overthinks decisions
- **Fix:** Set "good enough" standards, time-box decisions

**2. The Speed Demon**
- Rushes through everything
- Creates rework
- **Fix:** Slow down, quality checklists

**3. The Supervisor**
- Micromanages everything
- Can't let go
- **Fix:** Document processes, trust but verify

**4. The Saver**
- Hoards tasks to "save money"
- Does everything themselves
- **Fix:** Calculate true cost of your time

**5. The Self-Medicator**
- Uses busyness to avoid real issues
- Escapes into work
- **Fix:** Face the real problems, get support

### The Camcorder Method

How to document processes for delegation:

**Step 1: Record yourself doing the task**
- Screen share + audio
- Talk through decisions
- Include edge cases

**Step 2: Create SOP from recording**
- Write step-by-step
- Include screenshots
- Note decision points

**Step 3: Test with new person**
- Have them follow the SOP
- Note questions/confusion
- Update documentation

**Step 4: Iterate**
- Improve based on feedback
- Add FAQs
- Create video training

### Perfect Week Design

Structure your calendar for maximum effectiveness:

**Theme Days:**
| Day | Focus |
|-----|-------|
| Monday | Admin / Planning |
| Tuesday | Content / Marketing |
| Wednesday | Sales / Clients |
| Thursday | Product / Team |
| Friday | Strategy / Vision |

**Energy Mapping:**
- Morning (high energy): Creative work, strategy
- Afternoon (medium): Meetings, collaboration
- Evening (low): Admin, email

**Time Blocking Rules:**
1. Block production time FIRST
2. Batch similar activities
3. Buffer between meetings
4. Protect mornings ruthlessly
5. No meeting Mondays (or Fridays)

## SaaS-Specific Frameworks

### The SaaS Metrics That Matter

| Metric | Target | Why |
|--------|--------|-----|
| **MRR Growth** | 10%+ MoM | Predictable revenue |
| **Churn Rate** | <5% monthly | Retention focus |
| **LTV:CAC** | >3:1 | Profitable acquisition |
| **Time to Value** | <30 min | Activation focus |
| **NPS** | >50 | Customer happiness |

### The 3 Stages of SaaS Growth

**Stage 1: Problem-Solution Fit**
- Focus: Find 10 customers who love it
- Metrics: Retention, feedback
- Hiring: Don't

**Stage 2: Product-Market Fit**
- Focus: Repeatable sales process
- Metrics: MRR, conversion rates
- Hiring: Sales, support

**Stage 3: Scale**
- Focus: Efficiency, systems
- Metrics: Unit economics
- Hiring: Management, specialists
